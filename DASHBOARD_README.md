# ANZHFR Patient Information Dashboard

A simple, user-friendly dashboard for viewing hip fracture care statistics based on patient characteristics.

## Quick Start

### Option 1: Using a Local Web Server (Recommended)

1. **Convert the data to JSON** (if not already done):
   ```bash
   python3 convert_data_to_json.py
   ```

2. **Start a local web server**:
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Or Python 2
   python -m SimpleHTTPServer 8000
   ```

3. **Open your browser** and navigate to:
   ```
   http://localhost:8000/dashboard.html
   ```

### Option 2: Direct File Opening

If you're using Chrome or Edge, you can try opening `dashboard.html` directly, but you may encounter CORS issues. If you see errors, use Option 1 instead.

## Features

- **Personalized Information**: Enter your age, gender, cognitive status, and residence to see statistics for similar patients
- **Key Demographics**: View median age, gender distribution, and cognitive status
- **Care Journey Timeline**: See the main steps in hip fracture care
- **Outcome Measures**: 
  - Average length of stay
  - Percentage receiving rehabilitation
  - Percentage requiring new residential care placement
  - Surgery timing (within 48 hours)
  - Mortality rates (30-day, 90-day, 120-day)
- **Visualizations**: Interactive charts showing outcomes and discharge destinations

## How to Use

1. Fill in the form with your information:
   - **Age**: Your current age
   - **Gender**: Select your gender
   - **Cognitive Status**: Normal cognition or impaired/dementia
   - **Usual Residence**: Private home or residential care
   - **Walking Ability** (optional): Your pre-fracture walking ability

2. Click **"Get My Information"** button

3. View the results:
   - Demographics summary for similar patients
   - Key outcomes and statistics
   - Care journey timeline
   - Visual charts

## Technical Details

- **Frontend**: HTML, CSS, JavaScript
- **Charts**: Chart.js library
- **Data Format**: JSON (converted from CSV)
- **Browser Compatibility**: Modern browsers (Chrome, Firefox, Safari, Edge)

## Files

- `dashboard.html` - Main dashboard file
- `data.json` - Processed data file (generated by convert_data_to_json.py)
- `convert_data_to_json.py` - Script to convert CSV to JSON
